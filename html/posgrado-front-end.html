Posgrado Front-end

Este proyecto de diseño web tiene como objetivo crear una interfaz de una web-app para la gestión de proyectos. Utilizando tecnologías como HTML5, CSS3, JavaScript y Vue.js, se desarrollará una estructura escalable y adaptable a diferentes dispositivos. El enfoque principal es la usabilidad y el diseño visual atractivo, garantizando que los usuarios puedan interactuar de manera intuitiva con la plataforma. Además, se implementarán principios de accesibilidad y optimización de rendimiento para asegurar una carga rápida y eficiente.
